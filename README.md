# 📊 HR Analytics Dashboard | Power BI

## 📌 Overview

This project presents an interactive **HR Analytics Dashboard** built using Power BI to analyze employee data and uncover insights related to attrition, workforce demographics, and salary trends. The dashboard helps HR teams make data-driven decisions to improve employee retention and organizational efficiency.

---

## 🎯 Objectives

* Analyze employee attrition patterns
* Identify high-risk groups based on demographics and salary
* Track key HR metrics for better decision-making
* Provide actionable insights to improve retention strategies

---

## 📈 Key Features

* **Attrition Analysis:** Breakdown by age, gender, education, and job role
* **Salary Insights:** Attrition trends across different salary slabs
* **Workforce Metrics:** Total employees, attrition rate, average age, tenure
* **Department Insights:** Attrition across departments and roles
* **Interactive Filters:** Dynamic filtering for deeper analysis

---

## 🛠️ Tools & Technologies

* Power BI
* Data Cleaning & Transformation
* Data Visualization

---

## 📊 Key Insights

* Higher attrition observed in **26–35 age group**
* Employees in **lower salary brackets** show higher churn
* Certain job roles (e.g., Sales, Lab Technicians) have higher attrition
* Attrition tends to be higher in **early years of employment**

---

## 🚀 Outcome

The dashboard enables HR teams to quickly identify patterns in employee turnover and take proactive steps to improve retention and workforce planning.
